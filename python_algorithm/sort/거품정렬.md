# 거품정렬 (Bubble Sort)

## 1. 정의

연속된 두개의 숫자를 비교하면서 자리를 바꾸는 과정을 반복하여 정렬하는 알고리즘

**시간복잡도** : O(n<sup>2</sup>)

## 2. 구현

```python
def bubbleSort(arr):
    size = len(arr)
    for i in range(size):
        for j in range(1,size-i):
            if arr[j-1] > arr[j]:
                temp = arr[j-1]
                arr[j-1] = arr[j]
                arr[j] = temp
    return arr 
```
